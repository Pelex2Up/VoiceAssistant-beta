ИНСТРУКЦИЯ ПО ЗАПУСКУ KRIS_V2
========================================================================================================================

1. Перед запуском голосового ассистента Kris_v2 убедитесь, что установили все нужные для него библиотеки из файла
requirements.txt

2. В PyCharm IDE откройте файл main.py и запускайте код из самого IDE.

3. Ассистент реагирует только на команды, которые начинаются с его имени: "Крис, привет",
"Крис, расскажи что ты умеешь" (ассистент расскажет что он умеет делать).

4. Если ассистент вас не понял, повторите комманду. Если продолжает не реагировать, то значит команда не добавлена в
перечень того, что он умеет.
